# Word Table Processor

Desktop-приложение для обработки таблиц в Word-документах с российскими сметными нормативами.

## Функции

- **Обновление цен в Word-таблицах** — замена значений в документах на основе CSV-данных
- **Генерация CSV из Excel** — формирование файлов для обновления цен
- **Поддержка нормативов:**
  - ФЕР-2020 (ФССЦ, ФСЭМ)
  - ТЕР-2014 (ТССЦ, ТСЭМ)
  - ТЕР-2010 (ТССЦ, ТСЭМ)

## Требования

- Windows 10/11 (x64)
- .NET 6.0 Runtime (включён в дистрибутив)

## Установка

Скачайте `WordTableProcessor.exe` из папки `publish/` и запустите.

## Сборка из исходников

```bash
dotnet build
dotnet run
```

## Публикация

```bash
dotnet publish -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true -o ./publish
```
